Question-1: What are the different ways to select an element in the DOM?

getElementById(): Selects an element by its ID.
getElementsByClassName(): Selects all elements that have a specific class name.
getElementsByTagName(): Selects all elements with a specific tag name.
querySelector(): Uses CSS selectors to select the first matching element.
querySelectorAll(): Uses CSS selectors to select all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets or sets the HTML content inside an element. If you use innerHTML to set HTML, it will render the HTML as part of the document.
innerText: Gets or sets the text content of an element and its descendants, but unlike innerHTML, it respects the visibility of elements. innerText will not include text from hidden elements, and it also converts HTML into human-readable text.
textContent: Gets or sets the text content of an element and its descendants. It includes the text in all elements, regardless of styling, but does not parse any HTML tags. It treats all HTML as plain text.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where you use a single event listener on a parent element to manage events from multiple child elements. Instead of adding an event listener to each child, you add one to the parent and use it to handle events that bubble up from the children. This makes your code cleaner and can improve performance, especially with a large number of elements.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM (Document Object Model) is a method of event propagation where if an event is triggered on an element, it will then propagate (bubble) up through its ancestors in the hierarchy. This means the event is first handled by the innermost element and then successively by each of its parent elements up to the root. This behavior allows for more generalized event handling, where a single handler on a parent element can capture events from any of its child elements, reducing the need to attach event handlers to individual nodes.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, manipulation of the Document Object Model allows dynamic changes to the content, structure, and style of a webpage. This manipulation is achieved through methods provided by the DOM API that enable developers to create, add, or remove elements programmatically.
